# Online Grocery Billing System

## Description
A Java console-based application that generates grocery bills by calculating item prices, quantities, and the total amount.

## Features
- Add multiple grocery items
- Enter price and quantity
- Calculate total bill
- Display formatted bill

## Technologies Used
- Java
- VS Code

## How to Run
1. Compile the program:
   javac GroceryBilling.java
2. Run the program:
   java GroceryBilling
